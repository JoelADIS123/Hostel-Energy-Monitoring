# Hostel-Energy-Monitoring
Hostel Energy Monitoring Website using Cloud Computing concepts and frontend development

The project has AWS Lambda functions , API , DynamoDB and Eventbridge already integrated into the project.

To run the project , two options : 

Option 1:
Upload on AWS S3 bucket and run the object URL on that said S3 Object and run it on your browser 

Option 2 ( better):
Use the already created Object Invoke URL :  https://hostel-energy-dashboard-project.s3.eu-north-1.amazonaws.com/newerindex.html
